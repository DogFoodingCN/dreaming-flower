# Visual Constraints

## Visual Direction

The homepage should feel like a galaxy entrance: spatial, orbit-like, layered, and exploratory.

Suggested visual vocabulary:

- galaxy
- stars
- orbit
- nodes
- entrance
- space
- constellation

## Homepage Composition

- The center should communicate Dreaming Flower / DF identity.
- Future modules can appear as stars, planets, orbit nodes, or constellation points.
- Interactions should make the entrance feel alive without hiding essential content.

## Responsiveness

- Mobile layout must remain readable and touch-friendly.
- Desktop layout can use a wider galaxy composition.
- Hover effects must not be the only way to understand or access content.

## Theme

- The site must support day and night modes.
- Night mode may use a deeper galaxy atmosphere.
- Day mode should remain bright, legible, and aligned with the same visual system.
- Root viewport background, page backgrounds, glass panels, borders, and interactive states should all follow the active theme.
- Future module entries should reuse the same galaxy, constellation, orbit, and glass vocabulary instead of introducing unrelated visual systems.

## Motion

- Motion should be slow, calm, and spatial.
- Avoid heavy animation that distracts from the site identity or hurts readability.
- Respect basic accessibility: text remains readable, controls remain usable, and interaction states are visible.
